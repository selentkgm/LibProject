# LibRoject

>With this project, you can download videos from Youtube, request a music name or generate a QR code.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install.

```bash
pip install pytube
```
```bash
pip install segno
```
```bash
pip install tkinter
```
